# Sudoku Solver по скриншоту

Готовый Python-проект, который:
- берет скриншот поля судоку;
- находит сетку автоматически;
- распознает цифры;
- решает судоку;
- сохраняет итоговую картинку с дописанными цифрами;
- может автозаполнить клетки на сайте через мышку и клавиатуру.

## Структура проекта

```text
sudoku_solver_project/
├─ assets/
│  ├─ sample_input.png
│  └─ templates/
│     ├─ 1.png
│     ├─ 2.png
│     └─ ...
├─ output/
├─ src/
│  ├─ autofill.py
│  ├─ config.py
│  ├─ digit_reader.py
│  ├─ gui.py
│  ├─ image_processing.py
│  ├─ service.py
│  └─ solver.py
├─ tools/
│  └─ build_templates_from_sample.py
├─ main.py
├─ requirements.txt
├─ run_gui.bat
└─ run_sample.bat
```

## Как запустить

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Запусти GUI

```bash
python main.py gui
```

или просто двойным кликом по `run_gui.bat`.

## Быстрый тест на приложенном примере

```bash
python main.py solve --image assets/sample_input.png --output output/sample_result.png
```

или запусти `run_sample.bat`.

## Как пользоваться GUI

1. Нажми **Открыть изображение**.
2. Выбери скриншот судоку.
3. Нажми **Решить**.
4. Справа увидишь распознанное поле и готовое решение.
5. Нажми **Сохранить результат**, чтобы получить PNG с вписанными цифрами.

## Автозаполнение на сайте

После решения нажми **Автозаполнение на сайте**.

Программа спросит:
- `X` левого верхнего угла поля;
- `Y` левого верхнего угла поля;
- размер сетки по ширине в пикселях.

Дальше у тебя будет короткая пауза, после чего программа начнет:
- кликать по пустым клеткам;
- печатать найденные цифры.

Важно:
- сайт должен быть уже открыт;
- масштаб браузера должен быть стабильным;
- сетка не должна двигаться во время ввода.

## Как снять координаты для автозаполнения

Самый простой способ:
- открой судоку на сайте;
- сделай поле видимым целиком;
- в Paint, Lightshot или любом инструменте для скрина посмотри координаты верхнего левого угла;
- измерь ширину всей сетки.

## Как работает распознавание

По умолчанию проект использует **шаблоны цифр** из папки `assets/templates`.
Это надежнее для одного и того же сайта, где шрифт у цифр обычно не меняется.

Если шаблонный метод не уверен в ответе, включается запасной вариант через **Tesseract OCR**.

## Tesseract OCR

Для fallback-распознавания желательно установить Tesseract:
- Windows: установи `Tesseract OCR` и добавь его в `PATH`.

Если Tesseract не установлен, проект все равно сможет работать на похожих скринах того же сайта за счет шаблонов.

## Подгонка под другой сайт

Если у другого сайта другой шрифт цифр:
1. положи новый пример в `assets/sample_input.png`;
2. отредактируй координаты и цифры в `tools/build_templates_from_sample.py`;
3. запусти:

```bash
python tools/build_templates_from_sample.py
```

После этого в `assets/templates` создадутся новые шаблоны.

## Полезные команды

### Решить через консоль

```bash
python main.py solve --image path\to\your_sudoku.png --output output\result.png
```

### Решить и сразу заполнить на сайте

```bash
python main.py solve --image path\to\your_sudoku.png --autofill --top-left-x 100 --top-left-y 200 --grid-size 720
```
